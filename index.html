Starlark is a small and embeddable language designed for configuration and scripting. Starlark uses a Python-like syntax that should be familiar to most developers.

Why Starlark

  • Embeddable - Lightweight and easy to integrate into your application.
  • Simple - A minimalistic design prioritizing readability and ease of use for a wide audience.
  • Thread-Safe: Optimized for parallel execution without compromising safety. Data that is shared by multiple modules becomes immutable.
  • Deterministic Execution - Executing the same code twice will give the same results.
